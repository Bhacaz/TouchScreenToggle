TouchScreenToggle
=================

##Description##
Permet de faire tourner l'affichage de l'écran et la réaction tactile.

##Testé sur##
Lenovo ThinkPad Helix

Ubuntu Gnome 14.10

Gnome 3.14

##Installation##
Dans le répertoire, exécuté les commandes:

    $ sudo chmod +x setup.sh
    $ sudo ./setup.sh
